Preprocessing the data:

text <- pre_process_corpus(data, "text", replace_numbers = T,
                           root_gen = 'lemmatize', extra_stopwords = c("must", "can", "now",  
                                                                       "will","people"), output_corpus = T)
substr(text[[1]]$content, 1, 100)
## [1] "one privilege extend warm congratulation unite state delegation election presidency twentyfifth sess"
# build tokenizer
tokenizer <- function(x) {unlist(lapply(ngrams(words(x), 1), paste, collapse = " "), use.names = FALSE)}

# set lower bound
lbound <- round(0.015* length(text))

# create DTM
dtm1 <- as.matrix(DocumentTermMatrix(text, 
                          control = list(tokenize = tokenizer,
                                        bounds = list(global = c(lbound, Inf)))))

# create bigram DTM
tokenizer <- function(x) {unlist(lapply(ngrams(words(x), 2), paste, collapse = " "), use.names = FALSE)}

dtm2 <- as.matrix(DocumentTermMatrix(text, 
                          control = list(tokenize = tokenizer,
                                        bounds = list(global = c(lbound, Inf)))))

# create trigram DTM
tokenizer <- function(x) {unlist(lapply(ngrams(words(x), 3), paste, collapse = " "), use.names = FALSE)}

dtm3 <- as.matrix(DocumentTermMatrix(text, 
                          control = list(tokenize = tokenizer,
                                        bounds = list(global = c(lbound, Inf)))))
dtm <- cbind(dtm1, dtm2)
dtm <- cbind(dtm, dtm3)
dim(dtm)
## [1]     47 165174

freq_table <- data.frame(term = colnames(dtm), n = colSums(dtm),

5. Calculate the TF-IDF values for the DTM. In which years address does the term iraq provide the most semantic contribution? Evaluate the terms with the highest TF-IDF values in that address. Based on that finding, what do you infer that the speech was about?

number_of_docs <- nrow(dtm)
term_in_docs <- colSums(dtm > 0)
idf <- log(number_of_docs / term_in_docs)

# compute TF/IDF
tf_idf <- t(t(dtm) * idf)
names(tf_idf) <- colnames(dtm)
rownames(tf_idf) <- 1970:2016
interest = c("iraq")
tf_idf_interest <- tf_idf[, which(colnames(tf_idf) %in% interest)]

tf_idf_interest <- data.frame(tf_idf_interest)
tf_idf_interest$speech_no <- 1:nrow(tf_idf_interest)

tf_idf_interest <- tf_idf[, which(colnames(tf_idf) %in% c('nuclear', 'terrorist', 'freedom'))]

tf_idf_interest <- data.frame(year = 1970:2016, as.data.frame(tf_idf_interest))
tf_idf_interest$speech_no <- 1:nrow(tf_idf_interest)
df <- melt(tf_idf_interest, id.vars = c("year", "speech_no"), variable.name = "term", value.name = "tfidf")

ggplot(df, aes(x = year, y = tfidf, color = term)) + geom_line(linetype = 2) + geom_point()

Important observations: From above graph, it can be seen that the terms freedom, nuclear & terrorist appear to be dominating the distinct time period from year 1970 to year 2016. The term nuclear is much dominant between 1970-1980 meaning the speeches made in this time period had major focus on nuclear energy/ weapons/ war. After 1980 (till 1990), the TFIDF value for the term freedom is increasing. Speeches after 1990 are more focused on terrorism as the TFIDF of term terrorist is dominating after 1990.
